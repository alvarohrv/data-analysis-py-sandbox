## data-analysis-py-sandbox

Un entorno personal de estudio y experimentación para Python, análisis de datos, Machine Learning, ETL, visualización, y pruebas con herramientas como Jupyter, Docker, y bibliotecas del ecosistema científico.

´Este repositorio funciona como una sandbox:´ un espacio seguro donde probar ideas, practicar conceptos y documentar el proceso de aprendizaje sin preocuparse por la estructura de un proyecto de producción.

### Objetivos del repositorio
- Practicar análisis exploratorio de datos (EDA).
- Probar modelos básicos de machine learning.
- Explorar bibliotecas comunes de ciencia de datos:
numpy, pandas, matplotlib, seaborn, scikit-learn, etc.
- Organizar ejercicios y notas personales.
- Mantener un entorno reproducible usando Docker y requirements.txt.
